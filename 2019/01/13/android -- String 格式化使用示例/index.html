前言

这篇文章已经拖了快一个月。。虽然简单，还是总结一下吧。

最近做项目的时候遇到群发短信消息的需求，然后有个短信模板，大致格式如下 :

{userName} 在 {appName} 中对您发布了新的消息 ：{message} 快去查看吧。

也就是在创建短信消息体的时候会有 3 个变量 ，然后你估计会问，这有什么问题？ 嗯，确实没啥大问题。
所以我们这里来讨论的，是对于这个消息体字符串的创建我们有哪些方法，而哪个方法又显得比较优雅。
